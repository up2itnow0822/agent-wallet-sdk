# AgentWallet SDK — Feature Tiers

## Base (Free)

- **Agent Identity** — ERC-8004 Identity Registry (on-chain ERC-721 agent IDs)
- **Agent Reputation** — ERC-8004 Reputation Registry (scored feedback, summaries)
- **Agent Validation** — ERC-8004 Validation Registry (validator request/response)
- **ERC-6551 Token-Bound Accounts** — NFT-bound wallets with autonomous spending
- **Mutual Stake Escrow** — Reciprocal collateral task settlement between agents
- **Basic Optimistic Escrow** — Time-locked optimistic verification
- **x402 Payments** — HTTP 402 auto-pay protocol for agent-to-service payments
- **Multi-chain CCTP Bridge** — Circle CCTP V2 bridging across 17 chains + Solana
- **Spend Policies & Guards** — Per-token, per-period on-chain spending limits
- **Swap** — Uniswap V3 token swaps on Base/Arbitrum/Optimism
- **Fiat Onramp** — Opt-in fiat-to-crypto onramp integration
- **AP2** — Agent-to-Agent Protocol for task delegation and payment
- **Settlement** — On-chain settlement finalization
- **Gas Sponsorship** — ERC-4337 paymaster-based gas sponsorship
- **Solana Support** — Cross-chain Solana bridging via CCTP

## Premium

- **CowSwap Solver** — Submit batch auction solutions, earn COW tokens
- **Flash Executor** — Atomic flash loan execution for arbitrage and liquidations
- **MEV Protection** — Private mempool submission via Flashbots/MEV Blocker
- **Yield Staking** — Automated yield strategies (Aave V3, Compound V3, Morpho Blue)
- **Tax Reporting Module** — Transaction-level cost basis and gain/loss reporting
